Задание 2. Основы Python. Переменные и типы данных

### Описание
Напиши программу, которая запрашивает название книги, цену и количество экземпляров. Преобразуй цену к float, количество к int. Выведи чек с суммой заказа и укажи тип каждого значения.

### Пример входных данных
```
Python Tricks
899.5
3
```

### Пример выходных данных
```
Книга: Python Tricks (str)
Цена: 899.5 (float)
Количество: 3 (int)
Сумма заказа: 2698.5 (float)
```

### Критерии успешного выполнения
- Используются три вызова input() в указанном порядке.
- Цена преобразуется к float, количество к int.
- Сумма заказа считается как произведение цены на количество.
- В выводе присутствуют подписи, значения и типы.
- Программа завершается без ошибок при нормальном вводе.

### Подсказки
- Сохраняй значения в отдельных переменных, чтобы не запутаться.
- Для получения имени типа примени выражение type(значение).__name__.
- Используй f-строки для итогового вывода.

### Частые ошибки и ограничения
- Не переставляй порядок ввода данных.
- Не округляй результат самостоятельно.
- Не добавляй лишние строки в вывод.
