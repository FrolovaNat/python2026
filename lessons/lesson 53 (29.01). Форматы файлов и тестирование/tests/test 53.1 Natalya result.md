# Вопрос 1. Тема: Форматы JSON и CSV
  
**1. Что из перечисленного НЕ является допустимым типом данных в JSON?**
  
Варианты ответов:
1) ✅ Функция
2) Строка
3) Словарь
  
### ИИ анализ
JSON — это текстовый формат для обмена данными, который поддерживает строки, числа, логические значения, null, массивы и объекты (словари). Функции являются исполняемым кодом и не могут быть сериализованы в JSON, поэтому они не являются допустимым типом данных в этом формате. Ответ "Функция" выбран верно.

# Вопрос 2. Тема: Форматы JSON и CSV
  
**2. Какой параметр позволяет улучшить читаемость JSON-файлов при записи?**
  
Варианты ответов:
1) ✅ indent
2) formatter
3) newline
  
### ИИ анализ
Параметр `indent` в функции `json.dump()` или `json.dumps()` определяет количество пробелов для отступов, что форматирует JSON с вложенными структурами в удобочитаемом виде. Параметры `formatter` и `newline` не являются стандартными аргументами этих функций для улучшения читаемости. Выбор `indent` является правильным.

# Вопрос 3. Тема: Форматы JSON и CSV
  
**3. Какой метод используется для записи данных в JSON-файл?**
  
Варианты ответов:
1) ✅ json.dump()
2) json.save()
3) json.write()
  
### ИИ анализ
Для записи Python-объекта в JSON-файл используется функция `json.dump()`, которая принимает объект и файловый дескриптор. Методы `json.save()` и `json.write()` не существуют в стандартной библиотеке Python. Ответ выбран верно.

# Вопрос 4. Тема: Форматы JSON и CSV
  
**4. Какой метод используется для записи данных в CSV-файл?**
  
Варианты ответов:
1) ❌ csv.dump()
2) csv.read()
3) csv.writerows()
  
### ИИ анализ
Метод `csv.writerows()` объекта writer используется для записи нескольких строк (списка списков) в CSV-файл. Метод `csv.dump()` не существует, а `csv.read()` предназначен для чтения данных. Выбранный вариант неверен, правильным ответом является `csv.writerows()`.

# Вопрос 5. Тема: Форматы JSON и CSV
  
**5. Какой из следующих форматов поддерживает вложенные структуры данных?**
  
Варианты ответов:
1) CSV
2) ✅ JSON
3) Оба
  
### ИИ анализ
JSON изначально поддерживает вложенные структуры через объекты и массивы, что позволяет создавать сложные иерархические данные. CSV представляет данные в виде плоской таблицы и не имеет стандартной поддержки для вложенности. Поэтому верным является только JSON.

**6. Выберите верные утверждения о JSON (выбор нескольких вариантов):**
  
Варианты ответов:
1) JSON можно использовать только в Python
2) JSON не поддерживает тип данных null
