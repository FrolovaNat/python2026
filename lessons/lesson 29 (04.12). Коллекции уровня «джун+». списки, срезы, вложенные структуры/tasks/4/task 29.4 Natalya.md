# Задание 29.4. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Вложенные структуры. Учимся работать со списками словарей и аккуратно извлекать нужные данные из вложенных элементов.

## Условие задачи
Ты оформляешь каталог участников фермерской ярмарки:
1. Создай список словарей `vendors` с ключами `name`, `category`, `products` (список строк) и `price_level`.
2. Выведи сводку в формате `Имя (категория). уровень цен: ...`, перечислив товары через запятую.
3. Собери множество всех уникальных продуктов, проходя по каждому списку `products`.
4. Построй словарь `category -> количество участников`.
5. Отфильтруй продавцов с уровнем цен `"low"` и распечатай их товары отдельным списком.

## Пример входных данных
```python
vendors = [
    {"name": "Сады Данилы", "category": "овощи", "products": ["морковь", "свекла"], "price_level": "low"},
    {"name": "Пасека", "category": "мёд", "products": ["липовый", "гречишный"], "price_level": "medium"},
    {"name": "Сырная лавка", "category": "молочное", "products": ["бри", "качотта"], "price_level": "high"}
]
```

## Пример выходных данных
```
Сады Данилы (овощи). уровень цен: low. товары: морковь, свекла
Пасека (мёд). уровень цен: medium. товары: липовый, гречишный
Сырная лавка (молочное). уровень цен: high. товары: бри, качотта
Все продукты: {'качотта', 'морковь', 'бри', 'гречишный', 'липовый', 'свекла'}
Категории: {'овощи': 1, 'мёд': 1, 'молочное': 1}
Доступные продавцы: [{'name': 'Сады Данилы', 'products': ['морковь', 'свекла']}]
```

## Критерии успешного выполнения
- Список участников содержит словари с одинаковыми ключами
- Вывод по продавцам формируется циклом без жёстких индексов
- Уникальные продукты и подсчёт категорий получаются программно
- Фильтр по уровню цен не изменяет исходные данные

## Подсказки
- Для списка товаров используй `", ".join(products)`
- Множество удобно пополнять методом `update`
- Словарь подсчёта можно собирать через `setdefault` или проверку ключей

## Частые ошибки и ограничения
- Не редактируй список `vendors` при фильтрации
- Следи за правописанием ключей словаря
- Не копируй код для каждой категории. используй циклы
- Не дублируй продукты вручную. все уникальные значения собираются из данных
