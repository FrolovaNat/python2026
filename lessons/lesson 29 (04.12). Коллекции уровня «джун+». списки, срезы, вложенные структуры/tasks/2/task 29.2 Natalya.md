# Задание 29.2. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Индексация и срезы. Умеем выделять части списка, использовать отрицательные индексы и шаг для анализа данных без дополнительных циклов.

## Условие задачи
Ты ведешь список пульсовых зон за тренировку. Нужно:
1. Вывести первые пять значений, чтобы показать разогрев тренеру.
2. Получить средний участок с третьего по восьмой элементы.
3. Собрать каждый третий показатель для быстрой проверки стабильности.
4. Сформировать срез, который исключает первые два и последние два значения.

## Пример входных данных
```python
zones = [88, 92, 101, 109, 115, 118, 120, 117, 110, 100, 95]
```

## Пример выходных данных
```
Разогрев: [88, 92, 101, 109, 115]
Средняя часть: [101, 109, 115, 118, 120, 117]
Каждый третий: [101, 118, 100]
Без крайних: [101, 109, 115, 118, 120, 117, 110]
```

## Критерии успешного выполнения
- Все выборки сделаны исключительно срезами
- Используются разные комбинации start, stop и step
- Пустые границы применяются там, где это упрощает запись
- Каждый вывод подписан поясняющим текстом

## Подсказки
- Границы `[2:8]` дадут элементы с индекса 2 по 7
- Шаг `3` позволяет брать каждый третий элемент
- Чтобы отбросить края, поставь `zones[2:-2]`
- Не забывай, что `stop` не включается в результат

## Частые ошибки и ограничения
- Не задавай шаг 0, это вызовет исключение
- Убедись, что выбранные границы дают непустой список
- Не перезаписывай исходный список, работай только с копиями через срезы
- Не добавляй дополнительных циклов, все операции выполняются за счет синтаксиса `[:]`
