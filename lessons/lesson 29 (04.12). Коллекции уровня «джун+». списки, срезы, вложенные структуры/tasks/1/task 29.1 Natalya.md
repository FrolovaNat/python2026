# Задание 29.1. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Знакомство со списками. Разбираемся, как создавать списки, обращаться к элементам по индексу и безопасно их перебирать.

## Условие задачи
Тебе нужно составить список ингредиентов для быстрого обеда. Создай стартовый список, вставь полезные добавки в начало, увеличь список рекомендациями диетолога и распечатай длину списка. Затем покажи второй и последний элементы и выведи меню с нумерацией.

## Пример входных данных
```python
ingredients = ["макароны", "сыр", "томаты"]
addons = ["руккола", "оливковое масло"]
```

## Пример выходных данных
```
Всего ингредиентов: 5
Второй: сыр
Последний: оливковое масло
1. руккола
2. макароны
3. сыр
4. томаты
5. оливковое масло
```

## Критерии успешного выполнения
- Создан исходный список и добавлены новые элементы методами списка
- Количество элементов выводится через `len`
- Доступ к элементам выполнен по индексам, включая отрицательный
- Итоговый список выведен с нумерацией без пропусков

## Подсказки
- Используй `insert(0, ...)`, чтобы добавить элемент в начало
- Метод `extend` поможет быстро добавить несколько ингредиентов
- `enumerate` позволяет нумеровать элементы автоматически

## Частые ошибки и ограничения
- Не добавляй элементы через строковое сложение, иначе список превратится в строку
- Следи, чтобы список не пустел перед обращением к индексам
- Не выводи элементы в одной строке, лучше отдельными строками
- Избегай повторного применения `insert(0, ...)` внутри цикла перебора
