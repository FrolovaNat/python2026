# Задание 29.3. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Мутация и методы списков. Учимся добавлять, удалять и переставлять элементы, чтобы управлять списком ингредиентов без создания новых списков.

## Условие задачи
Ты готовишь список покупок для здорового обеда:
1. Создай стартовый список из трёх продуктов.
2. Добавь рекомендации нутрициолога в конец через `extend`.
3. Вставь «оливковое масло» на вторую позицию с помощью `insert`.
4. Удали продукт, который уже есть дома, методом `remove`.
5. Перемести последний элемент в середину списка (`pop` + `insert`).
6. Выведи итоговый список в порядке приготовления.

## Пример входных данных
```python
groceries = ["гречка", "курица", "овощи"]
coach = ["авокадо", "зелень"]
```

## Пример выходных данных
```
['гречка', 'оливковое масло', 'курица', 'овощи', 'зелень', 'авокадо']
```

## Критерии успешного выполнения
- Использованы минимум три разных метода мутации
- Есть пример вставки по индексу, удаления по значению и перемещения
- Изменения проводятся на оригинальном списке без копий
- Итоговый список печатается после всех операций

## Подсказки
- После `extend` можно сразу работать с обновлённым списком
- Индекс `1` соответствует второй позиции
- Перед `remove` убедись, что продукт присутствует
- Для переноса последнего элемента: `item = groceries.pop()` и `groceries.insert(len(groceries)//2, item)`

## Частые ошибки и ограничения
- Не модифицируй список внутри цикла вывода
- Избегай удаления элементов, которых нет в списке
- Следи за корректностью индексов при `insert`
- Не пересоздавай список через конкатенацию строк
