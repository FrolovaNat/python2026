### Блок 1: Работа со списками (закрепление)

Ты ведешь список книг для чтения. Создай список `books` с тремя книгами. Добавь две новые книги в конец списка. Выведи общее количество книг, первую и последнюю книгу. Затем выведи нумерованный список всех книг.

```python

new_books = ["1984", "Мастер и Маргарита"]
```

### Блок 2: Срезы (особое внимание)

У тебя есть список температур за неделю (в градусах Цельсия). 
1. Выведи первые три температуры с подписью "Начало недели".
2. Выведи последние три температуры с подписью "Конец недели".
3. Выведи каждую вторую температуру, начиная с первой, с подписью "Каждый второй день".
4. Выведи список без первого и последнего элементов с подписью "Средняя неделя".

Пример входных данных:
```python
temperatures = [15, 17, 16, 18, 19, 20, 18]
```

### Блок 3: Мутация списков

У тебя есть список дел на день. 
1. Добавь в конец списка дела: "позвонить маме" и "записаться к врачу".
2. Вставь в середину списка (после второго элемента) дело "проверить почту".
3. Удали из списка дело "купить хлеб" (если оно есть).
4. Перемести последнее дело в начало списка.

Пример входных данных:
```python
tasks = ["проснуться", "позавтракать", "купить хлеб", "поработать"]
```

### Блок 4: Вложенные структуры
