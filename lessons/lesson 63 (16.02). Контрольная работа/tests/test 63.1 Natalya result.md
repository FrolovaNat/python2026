Варианты ответов:
1) ✅ re
2) match
3) str
4) regex
5) text
  
### ИИ анализ
Стандартный модуль `re` в Python предоставляет полный набор функций для работы с регулярными выражениями, такие как `re.search()`, `re.match()`, `re.findall()` и `re.sub()`.  Он является основным и наиболее часто используемым инструментом для задач, связанных с шаблонным поиском в тексте.  

# Вопрос 31. Тема 5: Регулярные выражения
  
Что означает шаблон \d?
  
Варианты ответов:
1) Конец строки
2) ✅ Любая цифра
3) Любая буква
4) Точка
5) Любой пробел
  
### ИИ анализ
Специальная последовательность `\d` в регулярных выражениях соответствует любой десятичной цифре (эквивалентна классу символов `[0-9]`).  Это один из базовых шаблонов, часто используемых для извлечения чисел из текста.  Важно не путать его с `\D`, который соответствует любому символу, не являющемуся цифрой.  

# Вопрос 32. Тема 5: Регулярные выражения
  
Что означает символ . в регулярном выражении?
  
Варианты ответов:
1) Начало строки
2) Только точку
3) ✅ Любой один символ
4) Любую цифру
5) Только запятую
  
### ИИ анализ
Символ `.` (точка) в регулярных выражениях является метасимволом, который соответствует любому одному символу, за исключением символа новой строки (если не используется флаг `re.DOTALL`).  Чтобы найти именно символ точки, его необходимо экранировать обратным слэшем: `\.`.  

# Вопрос 33. Тема 5: Регулярные выражения
  
Что означает ^ в начале шаблона?
  
Варианты ответов:
1) Отрицание класса символов
2) Разделитель
3) Любой символ
4) Конец строки
5) ✅ Начало строки
  
### ИИ анализ
Метасимвол `^` (каретка) имеет два значения.  В начале шаблона (или подшаблона) он означает "начало строки" (или "начало текста" в зависимости от флагов).  Внутри квадратных скобок `[^...]` он означает отрицание, т.е. соответствует любому символу, не перечисленному в скобках.  В данном контексте вопроса верен первый вариант.  

# Вопрос 34. Тема 5: Регулярные выражения
  
Что означает $ в конце шаблона?
  
Варианты ответов:
1) ✅ Конец строки
2) Любая буква
3) Начало строки
4) Пробел
5) Любая цифра
  
### ИИ анализ
Метасимвол `$` соответствует концу строки (или концу текста до завершающего символа новой строки).  Он используется, чтобы указать, что шаблон должен заканчиваться в определенной позиции.  Например, шаблон `\d+$` найдет последовательность цифр только в том случае, если она расположена в самом конце строки.
# Вопрос 35. Тема 5: Регулярные выражения

Какая функция модуля re чаще всего проверяет, что строка начинается с шаблона?

Варианты ответов:
1) re.find()
2) re.parse()
3) ✅ re.match()
4) re.splitlines()
5) re.cut()

### ИИ анализ
Функция `re.match()` специально предназначена для проверки совпадения шаблона с начала строки.  Она возвращает объект совпадения, если шаблон найден в начале, или `None`, если нет.  Это отличает её от `re.search()`, которая ищет шаблон в любом месте строки.

# Вопрос 36. Тема 6: Основы асинхронного программирования

Что обычно означает «асинхронность» в программировании?

Варианты ответов:
1) ✅ Не ждать одну задачу
2) Только одно ядро
3) Только про файлы
4) Всегда быстрее
5) Нельзя тестировать

### ИИ анализ
Асинхронность означает, что программа может приостановить выполнение одной задачи, если она ожидает внешнего события (например, ответа от сети), и переключиться на выполнение другой задачи.  Это позволяет эффективнее использовать ресурсы, избегая простоев в ожидании, но не гарантирует, что код станет всегда быстрее.

# Вопрос 37. Тема 6: Основы асинхронного программирования

Для чего чаще всего полезен asyncio?

Варианты ответов:
1) Для рисования графики
2) ✅ Для задач с ожиданием
3) Для тяжелой математики
4) Для списков
5) Для JSON

### ИИ анализ
Модуль `asyncio` наиболее полезен для задач, связанных с вводом-выводом (I/O), где программа часто вынуждена ожидать: сетевые запросы, чтение файлов, взаимодействие с базами данных.  Он позволяет организовать конкурентное выполнение таких задач в одном потоке, не блокируя его на время ожидания.

# Вопрос 38. Тема 6: Основы асинхронного программирования

Как объявить асинхронную функцию в Python?

Варианты ответов:
1) async function f():
2) ✅ async def f():
3) def async f():
4) await def f():
5) def f() async:

### ИИ анализ
Асинхронная функция в Python объявляется с использованием ключевых слов `async def`.  Ключевое слово `async` ставится перед `def`, что указывает интерпретатору, что эта функция является корутиной и может содержать операции `await`.  Это стандартный синтаксис, введённый в PEP 492.

# Вопрос 39. Тема 6: Основы асинхронного программирования

Что делает await?

Варианты ответов:
1) Печатает результат
2) Завершает программу
3) Создает список
4) Импортирует модуль
5) ✅ Ожидает завершения асинхронной операции

### ИИ анализ
Ключевое слово `await` приостанавливает выполнение текущей корутины до тех пор, пока переданный ему awaitable-объект (например, другая корутина) не завершит свою работу.  Во время этой паузы управление возвращается в цикл событий (event loop), который может запустить другие задачи.  Это основа неблокирующего ожидания.

# Вопрос 40. Тема 6: Основы асинхронного программирования

Что такое «корутина» (coroutine) в Python?

Варианты ответов:
1) Функция async def
2) Тип списка
3) Ошибка кода
4) Число int
5) Файл настроек
6) ⚡ (0.2/1) Я не знаю ответа

### ИИ анализ
Корутина — это специальный тип функции, объявленный с помощью `async def`.  Её выполнение можно приостанавливать (с помощью `await`) и возобновлять позже, что позволяет организовывать конкурентность.  Корутина сама по себе является объектом, который нужно запустить (например, с помощью `asyncio.run` или `await`).

# Вопрос 41. Тема 6: Основы асинхронного программирования

Как чаще всего запускают главный asyncio-код в простых программах?

Варианты ответов:
1) asyncio.wait_all(main)
2) main(asyncio)
3) ✅ asyncio.run(main())
4) run.asyncio(main())
5) asyncio.start(main())

### ИИ анализ
Функция `asyncio.run()` является стандартным и рекомендуемым способом запуска главной корутины в программе.  Она создаёт цикл событий, запускает переданную корутину и корректно закрывает все асинхронные ресурсы по завершении.  Этот подход заменяет старые методы вроде `get_event_loop().run_until_complete()`.

# Вопрос 42. Тема 6: Основы асинхронное программирование

Что обычно значит «конкурентное выполнение» в asyncio?

Варианты ответов:
1) Всегда потоки
2) ❌ Пишут в один файл
3) Только в фоне
4) Всегда процессы
5) Задачи чередуются в ожидании

### ИИ анализ
В контексте asyncio конкурентное выполнение означает, что несколько задач выполняются, переключаясь между собой в одном потоке.  Переключение происходит в точках `await`, когда задача ожидает завершения операции ввода-вывода.  Это отличается от параллелизма (parallelism), где задачи выполняются одновременно на разных ядрах процессора.

1) ✅ SELECT
2) OPEN
3) FETCHALL

### ИИ анализ

# Вопрос 45. Тема 7: SQL, база и многотабличные запросы

Какой SQL-запрос обычно используют, чтобы добавить новую строку в таблицу?

Варианты ответов:
1) ADD
2) ✅ INSERT
3) APPEND
4) PUSH
5) CREATE

### ИИ анализ
Команда `INSERT INTO` используется для добавления одной или нескольких новых строк в таблицу.  Синтаксис включает указание имени таблицы, столбцов (опционально) и значений для вставки.  Например: `INSERT INTO users (name, age) VALUES ('Иван', 25);`.  `CREATE` используется для создания самой таблицы, а не для добавления данных в неё.

# Вопрос 46. Тема 7: SQL, база и многотабличные запросы

Какой SQL-запрос обычно используют, чтобы изменить существующие строки?

Варианты ответов:
1) EDIT
2) MODIFY
3) ALTER ROW
4) CHANGE
5) ✅ UPDATE

### ИИ анализ
Оператор `UPDATE` предназначен для модификации данных в существующих строках таблицы.  Он всегда используется вместе с `SET` для указания новых значений и часто с `WHERE` для точного определения, какие строки нужно изменить.  Без условия `WHERE` будут обновлены все строки в таблице, что может быть опасно.

# Вопрос 47. Тема 7: SQL, база и многотабличные запросы

Что делает WHERE в SQL?

Варианты ответов:
1) Фильтрует по условию
2) Создает таблицу
3) Удаляет таблицу
4) ❌ Сортирует строки
5) Объединяет строки

### ИИ анализ
Ключевое слово `WHERE` задаёт условие фильтрации для операторов `SELECT`, `UPDATE` и `DELETE`.  Оно ограничивает набор строк, над которыми выполняется операция, только теми, которые удовлетворяют указанному логическому выражению.  Для сортировки используется отдельное ключевое слово `ORDER BY`.

# Вопрос 48. Тема 7: SQL, база и многотабличные запросы

Для чего нужен JOIN?

Варианты ответов:
1) Переименовать таблицу
2) Создать базу
3) ✅ Соединить две таблицы
4) Удалить столбец
5) Посчитать сумму

### ИИ анализ
`JOIN` используется для объединения строк из двух или более таблиц на основе логической связи между ними.  Это фундаментальная операция в реляционных базах данных, позволяющая извлекать связанные данные, которые хранятся отдельно для нормализации.  Существуют разные типы JOIN: INNER, LEFT, RIGHT, FULL.

# Вопрос 49. Тема 7: SQL, база и многотабличные запросы

Что чаще всего указывают в условии соединения таблиц (ON) при JOIN?

Варианты ответов:
1) Два случайных числа
2) ✅ Равенство связанных полей
3) Название файла
4) Имя базы данных
5) Команда SELECT *

### ИИ анализ
В условии `ON` указывается предикат соединения, который чаще всего является равенством между столбцами из соединяемых таблиц (например, `table1.id = table2.user_id`).  Это условие определяет, как строки одной таблицы соотносятся со строками другой.  Соединение по равенству ключевых полей является самым распространённым типом соединения.
