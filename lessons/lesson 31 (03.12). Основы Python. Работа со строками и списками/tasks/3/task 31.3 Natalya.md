### Задание 31.3. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая помогает привести список покупок к аккуратному виду.

Требования:
- прочитать строку с покупками, разделенными запятой
- удалить пробелы вокруг каждого элемента
- привести все элементы к нижнему регистру
- убрать дубликаты
- отсортировать итоговый список по алфавиту

### Пример входных данных
```
  Хлеб, Молоко, хлеб,  сыр,  молоко 
```

### Пример выходных данных
```
['молоко', 'сыр', 'хлеб']
```

### Критерии успешного выполнения
- все элементы очищены от пробелов
- дубликаты удалены
- список отсортирован по алфавиту

### Подсказки
- используй `split(',')`, `strip()`, `lower()`
- для удаления дубликатов можно использовать `set`, но затем отсортировать результат

### Частые ошибки и ограничения
- не полагайся на порядок элементов после преобразования во множество
- учитывай возможность пустого ввода
