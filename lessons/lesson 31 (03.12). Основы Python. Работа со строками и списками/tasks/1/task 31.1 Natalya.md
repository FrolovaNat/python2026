### Задание 31.1. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая принимает строку с названием книги и приводит ее к единому формату.

Требования:
- убрать лишние пробелы по краям
- заменить несколько пробелов внутри строки на один
- привести каждое слово к виду `ПерваяБукваЗаглавная` остальные строчные
- вывести количество слов в названии и общую длину строки без лишних пробелов

### Пример входных данных
```
  вОЙна   и  мИР  
```

### Пример выходных данных
```
Название: Война И Мир
Слов: 3
Длина: 12
```

### Критерии успешного выполнения
- строка очищена от пробелов по краям
- лишние пробелы между словами уменьшены до одного
- каждое слово приведено к нужному регистру
- длина и количество слов посчитаны корректно

### Подсказки
- используй комбинацию `strip`, `split`, `' '.join(...)`
- для регистра можешь использовать `title()` или свой вариант

### Частые ошибки и ограничения
- не модифицируй строку по индексу . работай с новыми строками
- не забывай про крайний случай, когда введена пустая строка
