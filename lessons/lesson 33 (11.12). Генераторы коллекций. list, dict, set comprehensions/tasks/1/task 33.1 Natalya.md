# Задание 33.1. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Введение в генераторы коллекций» помогает научиться быстро замечать. где цикл можно заменить компактным и читаемым выражением.

## Условие задачи
**Сложность**: <span style="color: #f9a825">● средний</span>. примерное время: 15–20 минут

Тебе нужно обработать список имен сотрудников. Оставь только тех. у кого имя короче 6 символов. и получи список строк вида `"Имя — 4 буквы"`. Сформируй результат при помощи одного list comprehension.

## Пример входных данных
```
names = ["Антон", "Светлана", "Ира", "Борис", "Юля", "Даниил"]
```

## Пример выходных данных
```
['Антон — 5 букв', 'Ира — 3 буквы', 'Борис — 5 букв', 'Юля — 3 буквы']
```

## Критерии успешного выполнения
- Использован один list comprehension.
- Строки формируются через f-строку и содержат правильное количество букв.
- В список попадают только имена длиной < 6 символов.
- Сохраняется порядок исходного списка.

## Подсказки
- Напомни себе синтаксис `[выражение for имя in names if условие]`.
- Для подсчёта длины строки используй `len(name)`.
- Чтобы соблюдать падеж. заменяй слово «буквы» на «букв» при числе > 4.

## Частые ошибки и ограничения
- Не вставляй сложные условия внутрь выражения. держи его читаемым.
- Не используй `append`. результат должен создаваться comprehension-ом.
- Не изменяй исходный список `names`.
