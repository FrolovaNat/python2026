# Задание 33.2. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«List comprehensions» помогает быстро получить новый список с фильтрацией и преобразованием без промежуточных переменных.

## Условие задачи
**Сложность**: <span style="color: #f9a825">● средний</span>. примерное время: 15–20 минут

Есть список `meetings`, где каждый элемент — кортеж `(тема, участники, длительность)`. Нужно построить список строк `compact_meetings` с описанием только тех встреч, у которых длительность не больше 30 минут и участников меньше 5. Формат строки: `"<тема>: <участники> чел, <длительность> мин"`. Используй одно выражение list comprehension.

## Пример входных данных
```
meetings = [
    ("Старт проекта", 8, 25),
    ("Code review", 4, 30),
    ("Планирование", 3, 45),
    ("Демо", 2, 20)
]
```

## Пример выходных данных
```
['Code review: 4 чел, 30 мин', 'Демо: 2 чел, 20 мин']
```

## Критерии успешного выполнения
- Результат формируется единственным list comprehension.
- В список попадают только встречи с длительностью ≤ 30 и участниками < 5.
- Формат строки полностью совпадает с заданным.
- Исходный список `meetings` остаётся неизменным.

## Подсказки
- Используй распаковку: `for topic, members, minutes in meetings`.
- Фильтры ставь в конце выражения через `if`.
- Формат проще собрать f-строкой `f"{topic}: {members} чел, {minutes} мин"`.

## Частые ошибки и ограничения
- Не используй отдельные циклы и `append`.
- Следи, чтобы условие фильтрации проверяло оба критерия сразу.
- Не меняй порядок подходящих встреч.
