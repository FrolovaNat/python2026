2. `is_adult(age)` — возвращает `True` если `age >= 18`, иначе `False`.
3. `safe_div(a, b)` — возвращает `a / b`, но при `b = 0` возвращает `None`.

**Ключевые ограничения:**
- Чётко заданные входные параметры и ожидаемый тип вывода для каждой функции.
- Для `rectangle_info` обязателен кортеж из двух значений.
- Для `is_adult` строгая булева логика без `None`.
