### Отчёт по проверке решения задания 31.3

---

#### 1. Формулировка задания
**Цель**: Сравнить два множества растений (уже пересаженные в теплицу и оставшиеся в парке) и вывести:
- Общие растения (пересечение множеств)
- Растения, которые ещё нужно перевезти (элементы из второго множества, отсутствующие в первом).

**Ограничения**:
- Использовать операции над множествами (`&`, `-`, `|`)
- Не преобразовывать множества в списки без необходимости
- Вывод должен содержать пояснительные подписи
- Пример входных данных: `{'mint', 'lavender', 'rose'}`, `{'mint', 'maple', 'pine'}`
- Пример вывода:  
  ```
  Уже в обеих зонах: {'mint'}
  Нужно перевезти: {'pine', 'maple'}
  ```

---

#### 2. Результаты проверки
**Файл**: `lessons/lesson 31 (08.12). Словари и множества. паттерны использования, типичные ошибки/tasks/3/31.2.py`

```python
a = {'мята', 'лаванда', 'роза'}
b = {'мята', 'клён', 'сосна'}

print(f"В обеих зонах: {a&b}")
print(f"Нужно перевезти: {a-b}")
```

**Тестирование**:

- **Ошибка**: Вместо растений, которые нужно перевезти (элементы `b`, отсутствующие в `a`), выводятся элементы `a`, отсутствующие в `b`.

#### 4. Ошибки
**Блокирующая ошибка** (нарушение основной функциональности):
- **Неправильное вычисление растений для перевозки**:  
  Используется `a - b` вместо `b - a`, что приводит к выводу обратного результата.  
  **Исправление**:  
  ```python

  ```

  **Исправление**:  
  ```python
  print(f"Уже в обеих зонах: {a & b}")
  ```

#### 6. Полное решение
```python
set_a = {'mint', 'lavender', 'rose'}
set_b = {'mint', 'maple', 'pine'}

print(f"Уже в обеих зонах: {set_a & set_b}")
print(f"Нужно перевезти: {set_b - set_a}")
```

**Объяснение**:
- Пересечение (`&`) определяет общие растения.
- Разность `set_b - set_a` показывает элементы, которые есть в `set_b`, но отсутствуют в `set_a` (требуемые к перевозке).
