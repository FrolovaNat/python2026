### Отчёт по проверке задания 31.2

#### 1. Формулировка задания
**Цель**:  
Создать словарь `donations`, суммирующий пожертвования из списка `pledges`, затем безопасно удалить доноров из списка `finished` и вывести итоговый словарь.  

**Ключевые требования**:  
- Использовать `get`/`setdefault` для накопления сумм.  
- Удалять записи через `pop(key, запасное_значение)`.  
- Вывести обновлённый словарь после удаления.  
- Обрабатывать случаи, когда ключи из `finished` отсутствуют в `donations`.  

**Пример входных данных**:  
```python
pledges = [('anna', 300), ('pavel', 150), ('anna', 200), ('dasha', 400)]
finished = ['pavel']
```

**Ожидаемый вывод**:  
Обновлённый словарь (например, `{'anna': 500, 'dasha': 400}`).

---

#### 2. Результаты проверки
**Файл**: `31.2.py`  
**Код студента**:  
```python
pledges = [('anna', 300), ('pavel', 150), ('anna', 200), ('dasha', 400)]
finished = ['pavel']

donations = {}
for name,count in pledges:
    donations[name] = donations.get(name,0) + count
for y in finished:
    removed = donations.pop(y, 'вернул деньги')
    print(f"{y}: {removed}")
print(donations)
```

**Тестирование**:  
1. Запуск с примером входных данных → вывод:  
   ```
   pavel: 150
   {'anna': 500, 'dasha': 400}
   ```  
2. Добавление отсутствующего донора в `finished` (например, `finished = ['pavel', 'ivan']`) → вывод:  
   ```
   pavel: 150
   ivan: вернул деньги
   {'anna': 500, 'dasha': 400}
   ```  
   Ошибок не возникает.
