Пример входных данных:
```python
pledges = [

finished = ["pavel", "ivan"]
```

## Задача 2. Множества доменов email

На платформе есть список писем, каждое содержит email отправителя.

Пример входных данных:
```python
emails = [
    "user1@corp.ru",
    "user2@example.com",
    "user3@corp.ru",
    "user4@test.io",
]
trusted = {"corp.ru", "example.com"}
```

### Требуется
1. Собрать множество всех доменов, которые встречаются в списке `emails`.
2. Найти множество доменов, которые **не** входят в `trusted`.
3. Распечатать два множества с понятными подписями.

### Критерии
- Домены корректно отделяются от адреса (часть после `@`).
- Используются операции множеств (`-` или `difference`).

## Задача 3. Пересечения групп

Есть три учебные группы, каждая представлена множеством логинов.

Пример входных данных:
```python
a = {"natalya", "ivan", "olga"}
b = {"olga", "petr", "natalya"}
c = {"ivan", "sergey"}
```

### Требуется
1. Найти студентов, которые учатся и в `a`, и в `b`.
2. Найти студентов, которые есть в `a`, но отсутствуют в `c`.
3. Вывести результаты с подписями:
   ```text
   В группах A и B: {...}
   В A, но не в C: {...}
   ```

## Задача 4. Подготовка к comprehensions

Нужно подготовить статистику по темам уроков, которую удобно будет фильтровать.

Пример входных данных:
```python
topics = [
    ("dict", 4),
    ("set", 3),
    ("list", 5),
    ("dict", 2),
]
limit = 5
```

### Требуется
1. Собрать словарь `hours_by_topic`.
2. Сформировать новый словарь только для тем, где суммарное время не меньше `limit`.
3. Распечатать оба словаря.
